Description

In this stage, you should parse -sortIntegers argument. This argument should indicate that the numbers that user input should be sorted. 

If the -sortIntegers argument is presented, you should ignore other arguments, otherwise, the behavior of the program should be the same as in the previous stage.

After processing user data, print Sorted data: and then print all integers sorted in ascending order.

Example

Run configuration example:

java SortingTool -sortIntegers

Run example:

Input:

1 -2   33 4
42
1                 1

Output (for sorting task):

Total numbers: 7.
Sorted data: -2 1 1 1 4 33 42
